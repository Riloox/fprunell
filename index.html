Personal Projects

Proyectos Personales

AFK Timeskip for Minecraft 1.21

AFK Timeskip is a Minecraft plugin for Spigot 1.21 servers that enhances the AFK (Away From Keyboard) experience. When a player goes AFK, they become invulnerable. If they are the only player on the server, the plugin increases the server tick rate to accelerate time passage, simulating a "time skip" effect.

AFK Timeskip es un plugin de Minecraft para servidores Spigot 1.21 que mejora la experiencia AFK (Away From Keyboard). Cuando un jugador está AFK, se vuelve invulnerable. Si son el único jugador en el servidor, el plugin aumenta la velocidad del servidor para acelerar el paso del tiempo, simulando un efecto de «salto en el tiempo».

Survival Spawner Manager for Minecraft 1.21

It allows players to mine Mob Spawners with Silk Touch and modify the mob spawned and equipment of the mob spawned if applicable, by using player levels.

Permite a los jugadores minar Mob Spawners con Silk Touch y modificar el mob engendrado y el equipamiento del mob engendrado si procede, utilizando los niveles del jugador.

KeyListener

The KeyListener program is a Python-based utility that listens for specific keyboard input sequences and performs predefined actions. It is designed to run in the background and can be configured to execute custom actions when certain key combinations are pressed. The program also supports adding itself to Windows startup for persistent execution.

El programa KeyListener es una utilidad basada en Python que escucha secuencias específicas de entrada del teclado y realiza acciones predefinidas. Está diseñado para ejecutarse en segundo plano y puede configurarse para ejecutar acciones personalizadas cuando se presionan ciertas combinaciones de teclas. El programa también admite agregarse al inicio de Windows para una ejecución persistente.

Almacén básico de clave-valor en memoria con persistencia de archivos y cifrado, creado como proyecto personal de aprendizaje para explorar el funcionamiento de las bases de datos.
